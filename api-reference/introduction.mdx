The Indie Security API enables programmatic access to security assessments, findings, and reports.

## Base URL

```
https://api.indiesecurity.com/v1
```

## Authentication

All API requests require a Bearer token in the Authorization header:

```bash
curl -X GET 'https://api.indiesecurity.com/v1/targets' \
  -H 'Authorization: Bearer is_live_xxxxxxxxxxxxx'
```

<Warning>
  Keep your API keys secure. Never commit them to version control.
</Warning>

### Getting Your API Key

1. Navigate to **Dashboard** → **Settings** → **API Keys**
2. Click **Generate New Key**
3. Copy and store the key securely

<Info>
  API keys are scoped to your organization. All users share the same key.
</Info>

## Rate Limits

| Tier | Requests/Minute | Concurrent Scans |
|------|-----------------|------------------|
| Free | 60 | 1 |
| Pro | 300 | 5 |
| Enterprise | Unlimited | Unlimited |

Rate limit headers are included in all responses:

```
X-RateLimit-Limit: 300
X-RateLimit-Remaining: 299
X-RateLimit-Reset: 1640995200
```

## Error Codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `UNAUTHORIZED` | 401 | Invalid or missing API key |
| `FORBIDDEN` | 403 | Insufficient permissions |
| `NOT_FOUND` | 404 | Resource not found |
| `VALIDATION_ERROR` | 422 | Invalid request parameters |
| `RATE_LIMITED` | 429 | Too many requests |
| `SERVER_ERROR` | 500 | Internal server error |

## Quick Examples

<CodeGroup>
```python Python
from indiesecurity import Client

client = Client(api_key="is_live_xxx")

# List targets
targets = client.targets.list()

# Start assessment
assessment = client.assessments.create(
    target_id="target_123",
    mode="quick"
)

# Get findings
findings = client.findings.list(target_id="target_123")
```

```javascript Node.js
import { IndieSecurity } from '@indiesecurity/sdk';

const client = new IndieSecurity({ apiKey: 'is_live_xxx' });

// List targets
const targets = await client.targets.list();

// Start assessment
const assessment = await client.assessments.create({
  targetId: 'target_123',
  mode: 'quick'
});

// Get findings
const findings = await client.findings.list({ targetId: 'target_123' });
```

```bash cURL
# List targets
curl -X GET 'https://api.indiesecurity.com/v1/targets' \
  -H 'Authorization: Bearer is_live_xxx'

# Start assessment
curl -X POST 'https://api.indiesecurity.com/v1/assessments' \
  -H 'Authorization: Bearer is_live_xxx' \
  -H 'Content-Type: application/json' \
  -d '{"target_id": "target_123", "mode": "quick"}'
```
</CodeGroup>
